# ፁ Handwritten Digit Recognition using CNN

A real-time handwritten digit recognition web application built using a Convolutional Neural Network (CNN) trained on the MNIST dataset and deployed using Streamlit.

ፁ **GitHub Repository:** [https://github.com/M-Divya29/Hand-Written-Digit-Recognition](https://github.com/M-Divya29/Hand-Written-Digit-Recognition)

---

## ፁ Project Overview
This project implements a Deep Learning model that identifies handwritten digits from 0-9. It features an interactive drawing canvas where users can draw a digit and get an instant prediction with high confidence.

## ፁ Key Features
- ✏️ **Interactive Canvas**: Draw digits directly in the browser.
- ፁ **CNN Model**: High-accuracy deep learning architecture built with TensorFlow/Keras.
- ፁ **Preprocessing**: Automated image resizing and grayscale conversion.
- ፁ **Responsive UI**: Clean interface built with Streamlit.

## ፁ Tech Stack
- **Languages**: Python
- **Deep Learning**: TensorFlow, Keras
- **Web Framework**: Streamlit
- **Data Handling**: NumPy, Scikit-learn
- **Image Processing**: OpenCV

## ፁ How to Run Locally
1. **Clone the repo**:
   ```bash
   git clone https://github.com/M-Divya29/Hand-Written-Digit-Recognition.git
   cd Hand-Written-Digit-Recognition
   ```
2. **Install requirements**:
   ```bash
   pip install -r requirements.txt
   ```
3. **Run application**:
   ```bash
   streamlit run app.py
   ```
